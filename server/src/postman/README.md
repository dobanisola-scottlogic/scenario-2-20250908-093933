The collection in this file contains the backend API calls for the hackathon - any hackathon, team or game names required for future calls will be automatically generated. Environments exist for both when the hackathon is being run locally and when it is deployed to AWS. The default URL for AWS deployment is `https://hackathon.hackathon-sl.com` but can be changed depending on what image is being used; for example, using the image 'hactest' would need the URL `https://hactest.hackathon-sl.com`
